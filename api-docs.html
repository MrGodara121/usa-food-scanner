GET /api/search?q={query}&type={type}

Search products, categories, or ingredients

Parameters:

  • q - search query (required)
  • type - all, products, categories, ingredients, blog (optional, default: all)
  • limit - number of results (optional, default: 20)

GET /api/alternatives?barcode={barcode}

Get healthier alternatives for a product

GET /api/micronutrients?barcode={barcode}

Get detailed vitamin and mineral information

GET /api/adaptive-calories?user_id={userId}

Get personalized calorie recommendations based on weight trends

POST /api/image-scan

Upload a photo of food to identify product

Form Data:

  • image - image file (jpeg/png)
