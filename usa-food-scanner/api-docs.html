Authentication

Headers

All API requests require an API key in the header:

{
    "x-api-key": "YOUR_API_KEY",
    "Content-Type": "application/json"
}

Base URL

https://api.usafoodscanner.com/v1

Endpoints

GET /product?barcode={barcode}

Get detailed product information by barcode

Parameters:

  • barcode (required) - 12-digit UPC barcode

Response:

{
    "barcode": "0123456789012",
    "name": "Lay's Classic Chips",
    "brand": "Lay's",
    "health_score": 45,
    "grade": "D",
    "ingredients": [
        {"name": "Potatoes", "is_healthy": true},
        {"name": "Salt", "is_healthy": false}
    ],
    "macros": {
        "calories": 160,
        "protein": 2,
        "carbs": 15,
        "fat": 10
    },
    "micronutrients": {
        "vitamin_a": 0,
        "vitamin_c": 15,
        "calcium": 20,
        "iron": 0.5
    }
}

GET /search?q={query}&type={type}&limit={limit}

Search products, categories, or ingredients

Parameters:

  • q (required) - Search query
  • type (optional) - all/products/categories/ingredients/blog (default: all)
  • limit (optional) - Results per page (default: 20, max: 100)

GET /alternatives?barcode={barcode}

Get healthier alternatives for a product

GET /micronutrients?barcode={barcode}

Get detailed micronutrient data (84+ nutrients)

POST /scan

Track a product scan (for analytics)

Request Body:

{
    "barcode": "0123456789012",
    "user_id": "user_123"
}
